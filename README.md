## Overview

This assignment includes two main tasks:

- **Task 1: Number Analysis App**  
  Implements a program that analyzes a list of integers. It calculates the maximum, minimum, sum, and average values of the list. Bonus features include counting negative numbers, handling empty lists safely, and producing a sorted output.

- **Task 2: Async Calculator App**  
  Implements a Calculator class with synchronous arithmetic methods and an asynchronous wrapper that simulates network delays. Includes error handling for divide-by-zero and unknown operations. Bonus features include a history log, chained operations, custom exceptions, and parallel asynchronous calculations.
